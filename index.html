Guessing Game in VHDL

In this assignment the students had to program their Basys3 FPGA boards in order to make their boards serve as a Guessing Game. This game was designed in VHDL using Vivado editor. In this game there are 2 players. The first player enters any desired number ranging from 0000 to FFFF (Hex numbers) and the second player has to guess this number. If the guessed number is too high, The seven segment display shows a 'Too-High' sign indicating that the entered number is greater than the actual number. Similarly it also shows a 'Too-Low' sign if the entered number is less than the actual number. On a correct guess all the LEDS blink in celebration.
